### 数据处理

读取生成的评论文件，数据格式处理

```python
import jieba
file = open('comments.txt', encoding='utf-8')
txt = file.read()
# 分词
txt_list = jieba.lcut(txt)
txt_str = " ".join(txt_list)
```

```python
img = Image.open("sharp.png")
```
